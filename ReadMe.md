查找,预览,执行,重复执行,算是对tc命令的深度使用,支持中文快搜,和空格多个关键词搜索,搜索命令的menu部分

# V1.1 2024-2-1

增加类似vim的重复上一条命令功能,默认使用ctrl r,重复上一条执行的命令,tmd菜单中默认最后一次执行的命令置顶
功能说明:比如某些命令没有加快捷键和工具栏,临时需要用个10次,可以执行一次之后,后面连续9次ctrl r重复



# 基本使用方法
1. 拿来就用,无须填写路径
2. 呼出按钮有三种方案
- 鼠标中键跟随鼠标呼出
- F12出现顶部
- win enter跟随光标出现
可以在配置文件中修改
**注意默认呼出改成\**

最基本使用例子:呼出搜索框,搜索desk,选择命令,enter执行,完毕

## 主要功能
1. cm和em都支持模糊搜索,比如输入desk zm op cd 匹配的是cd到桌面的命令,顺序错,位置错一样能搜到
2. ctrl d钉住和取消钉,不输入的时候显示钉住的命令
3. :进入cm命令搜索模式
4. ctrl c复制全部,ctrl 1,2,3对应复制消息号,menu等自行看settings
5. 回车执行em或者cm命令
6. 通过预览窗口可以查询命令信息
7. 增加LanguageEM命令添加功能,具体我也不知道,你要是不知道说明你不需要,不要动那个参数
8. 可以设置正则规则不显示某些菜单,预设的是#开头不匹配,自行写正则
9. 配置项修改自动应用,无须手动重启
10.快捷键自定义,预览位置自定义,预览开关一个键开关
11. 增加编辑器快速打开usercmd定位命令手动修改默认快捷键ctrl e,可以指定是否模拟按键定位,没写地址按下会让你填写地址

**2024-2-1重磅新功能,时隔一年更新重复上一条命令功能**

vim可以重复执行上一条命令,如今tmd也支持重复执行上一条命令,比如tmd中搜索了解散文件夹命令,执行之后,可以通过设置好的重复快捷键,反复调用这个命令,比如我设置的是ctrl r,就可以一直重复这个命令

适合一些平常不咋用,没有配置快捷键和工具栏,临时需要用很多次的情况

**两种重复的方法,第一种就是设置快捷键,第二种就是每次打开tmd后,置顶的第一条永远是上次执行的命令,可以直接enter执行**

# 自启
## 自动配置
需要自启的话,运行autorun写入tc自启,如果路径不对会退出

## 手动
添加删除语句到autorun.cfg可以开关自启
ShellExec "%COMMANDER_PATH%\TMD\TMD.exe"
